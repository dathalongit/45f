# css transition
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        .target1 {
            width: 100px;
            height: 150px;
            background-color: green;
            color: white;
            transition: width 2s, height 3s, background-color 2s;
        }

        .target1:hover {
            width: 200px;
            height: 300px;
            background-color: orange;
        }
    </style>
</head>
<body>

    <pre>
        css transition
    </pre>

    <div class="target1">
        #target1
    </div>
</body>
</html>
```

#  css transform
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        .target1 {
            width: 200px;
            height: 200px;
            background-color: green;
        }

        .target1:hover {
            transform: translate(50px, 100px);
        }
    </style>
</head>
<body>

    <pre>
        css transform
        translate() dịch chuyển so với vị trí ban đầu
    </pre>

    <div class="target1">
        .target1
    </div>

</body>
</html>
```
