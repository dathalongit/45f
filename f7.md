# flexbox

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        * {
            margin: 0;
            padding: 0;
        }

        .banner {
            background-color: orange;
            padding: 20px;
            display: flex;
        }

        .banner-item {
            background-color: green;
            padding: 20px;
            margin: 20px;
        }
    </style>
</head>
<body>

    <h1>Flexbox</h1>

    <pre>
        1 - flex container : hiểu đơn giản là khung chứa bên ngoài
        2 - flex item : hiểu đơn giản là thẻ con trực tiếp bên
        trong container

        ví dụ :
        .banner là flex container
        .banner-item là flex item
    </pre>

    <div class="banner">
        <div class="banner-item">1</div>
        <div class="banner-item">2</div>
        <div class="banner-item">3</div>
        <div class="banner-item">4</div>
        <div class="banner-item">5</div>
    </div>



</body>
</html>
```

# flex direction

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        * {
            margin: 0;
            padding: 0;
        }

        .banner {
            background-color: orange;
            padding: 20px;
            display: flex;
            flex-direction: column;
            /* các flex item bên trong nằm chồng lên nhau
            như trạng thái mặc định khi không sử dụng flex */
        }

        .banner-item {
            background-color: green;
            padding: 20px;
            margin: 20px;
        }
    </style>
</head>
<body>

    <h1>Flexbox flex-direction: column;</h1>

    <pre>
        1 - flex container : hiểu đơn giản là khung chứa bên ngoài
        2 - flex item : hiểu đơn giản là thẻ con trực tiếp bên
        trong container

        ví dụ :
        .banner là flex container
        .banner-item là flex item

        trong flex container có 6 thuộc tính quan trọng :
        flex-direction : chỉ định hướng sắp xếp của các flex item bên trong
        flex-wrap
        flex-flow
        justify-content
        align-items
        align-content
    </pre>

    <div class="banner">
        <div class="banner-item">1</div>
        <div class="banner-item">2</div>
        <div class="banner-item">3</div>
        <div class="banner-item">4</div>
        <div class="banner-item">5</div>
    </div>



</body>
</html>
```
