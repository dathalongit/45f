# flexbox

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        * {
            margin: 0;
            padding: 0;
        }

        .banner {
            background-color: orange;
            padding: 20px;
            display: flex;
        }

        .banner-item {
            background-color: green;
            padding: 20px;
            margin: 20px;
        }
    </style>
</head>
<body>

    <h1>Flexbox</h1>

    <pre>
        1 - flex container : hiểu đơn giản là khung chứa bên ngoài
        2 - flex item : hiểu đơn giản là thẻ con trực tiếp bên
        trong container

        ví dụ :
        .banner là flex container
        .banner-item là flex item
    </pre>

    <div class="banner">
        <div class="banner-item">1</div>
        <div class="banner-item">2</div>
        <div class="banner-item">3</div>
        <div class="banner-item">4</div>
        <div class="banner-item">5</div>
    </div>



</body>
</html>
```

# flex direction

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        * {
            margin: 0;
            padding: 0;
        }

        .banner {
            background-color: orange;
            padding: 20px;
            display: flex;
            flex-direction: column;
            /* các flex item bên trong nằm chồng lên nhau
            như trạng thái mặc định khi không sử dụng flex */
        }

        .banner-item {
            background-color: green;
            padding: 20px;
            margin: 20px;
        }
    </style>
</head>
<body>

    <h1>Flexbox flex-direction: column;</h1>

    <pre>
        1 - flex container : hiểu đơn giản là khung chứa bên ngoài
        2 - flex item : hiểu đơn giản là thẻ con trực tiếp bên
        trong container

        ví dụ :
        .banner là flex container
        .banner-item là flex item

        trong flex container có 6 thuộc tính quan trọng :
        flex-direction : chỉ định hướng sắp xếp của các flex item bên trong
        flex-wrap
        flex-flow
        justify-content
        align-items
        align-content
    </pre>

    <div class="banner">
        <div class="banner-item">1</div>
        <div class="banner-item">2</div>
        <div class="banner-item">3</div>
        <div class="banner-item">4</div>
        <div class="banner-item">5</div>
    </div>



</body>
</html>
```

#  flex-wrap áp dụng cho flex container
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style type="text/css">
        .flex-container {
            display: flex;
            flex-wrap: wrap;
            /* có bao chứa tất cả các flex item */
            background-color: green;
        }

        .flex-container > div {
            padding: 20px;
            margin: 20px;
            font-size: 18px;
            background-color: yellowgreen;
        }
    </style>
</head>
<body>

    <pre>
        flex-wrap áp dụng cho flex container
        quyết định flex container có bao chứa được các thành phần con hay không
    </pre>

    <div class="flex-container">
        <div>1</div>
        <div>2</div>
        <div>3</div>
        <div>4</div>
        <div>5</div>
        <div>6</div>
        <div>7</div>
        <div>8</div>
        <div>9</div>
        <div>10</div>
        <div>11</div>
        <div>12</div>
    </div>

</body>
</html>
```

# flex-flow
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style type="text/css">
        .flex-container {
            display: flex;
            flex-direction: column;
            flex-wrap: wrap;
            background-color: green;
        }

        .flex-container > div {
            padding: 20px;
            margin: 20px;
            font-size: 18px;
            background-color: yellowgreen;
        }
    </style>
</head>
<body>

    <pre>
        flex-flow là cách viết rút gọn của flex-direction và
        flex-wrap

        flex-flow: A B;
        trong đó A là giá trị flex-direction
        B là giá trị của flex-wrap

    </pre>

    <div class="flex-container">
```
